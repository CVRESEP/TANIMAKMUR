Panduan Penggunaan
Cara menjalankan aplikasi dengan database lokal PC
  1. Jalankan server — Di terminal, ketik: node server.js
  2. Buka halaman ini — Akses http://localhost:3737/migrate.html di browser yang sama dengan data lama
  3. Klik Mulai Migrasi — Data dari localStorage akan dikirim ke SQLite di PC
  4. Gunakan Dashboard — Buka http://localhost:3737/dashboard.html untuk akses penuh
  5. Database tersimpan di data/tanimakmur.db — Aman dari hapus cache browser!
