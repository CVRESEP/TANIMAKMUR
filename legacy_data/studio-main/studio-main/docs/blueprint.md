# **App Name**: TANI MAKMUR

## Core Features:

- Dashboard: Displays key summaries of operations, such as total redemptions, distributions and remaining balances.
- Product Data: Displays list of all products.
- Kiosk Directory: Displays a directory of kiosk information.
- Daily Summary: Daily summary of redemptions, DO expenses, and distributions in tons. This data is extracted using an AI tool, helping identify patterns.
- Generate Reports: Generate simple harian reports. Uses an AI tool to generate concise report summaries.

## Style Guidelines:

- White background provides a clean and modern aesthetic.
- Light blue (#ADD8E6) for primary elements and accents creates a sense of calm and trustworthiness.
- 'Roboto' (sans-serif) provides a modern and readable style suitable for both headlines and body text. It ensures clarity across the dashboard and data displays.
- Simple, clear icons representing different products, kiosks, and actions. Use a consistent style for easy recognition.
- A clean and organized layout that emphasizes key data and workflows. The dashboard uses clear sections, and data tables are easy to read.
